Narrated MP4 walkthrough

Same Acme contact flow as the first demo, but the output is a single narrated video instead of an interactive page. Per-step CSS spotlights highlight whatever the narrator is talking about; AWS Polly (Arthur, en-GB, neural) reads each step; ffmpeg composites the title + caption bars in post and stitches the clips.
1 · CapturePlaywright drives the app, applies spotlight CSS, screenshots clean 1920×1080 frames.
2 · NarrateAWS Polly synthesizes one MP3 per step using the Arthur (en-GB) neural voice.
3 · OverlayA headless Playwright pass renders title + caption bar PNGs from a tiny HTML template.
4 · Stitchffmpeg composites each still + both overlays + the matching MP3, then concatenates.
Regenerate with npm run video (wrapped in aws-vault exec for Polly creds).

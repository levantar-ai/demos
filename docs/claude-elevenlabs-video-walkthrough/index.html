Localized narrated walkthrough

The same Acme contact flow as the Polly demo, now in four languages from one source script. Claude translates each step (narration, captions and UI labels); ElevenLabs (George, a British voice, multilingual model) speaks every language; ffmpeg stitches one MP4 per locale, each with its flag dropped into the title bar in post. Pick a language:
1 · Capture Playwright drives the app, applies spotlight CSS, screenshots clean 1920×1080 frames. Shared across every language.
2 · Translate Claude translates each step (narration, captions, UI labels) into French, Spanish and Japanese.
3 · Narrate ElevenLabs synthesizes one MP3 per step in the George voice. A single multilingual model covers all four languages.
4 · Stitch ffmpeg composites each still, the title and caption bars and a per-locale flag, then concatenates one MP4 per language.
Regenerate with npm run localize then narrate / overlays / render per language (needs an ELEVENLABS_API_KEY).
